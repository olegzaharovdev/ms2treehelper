# ms2treehelper
Массовая отметка категорий в окне создания и редактирования опций miniShop2.  Компонент добавляет в дерево категорий miniShop2 компактные кнопки массовых действий прямо в штатный toolbar: отметить все категории, отметить все дочерние категории выбранного элемента и снять отметки со всех категорий.
