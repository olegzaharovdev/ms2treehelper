# ms2TreeHelper

`ms2TreeHelper` — дополнительный компонент для **MODX Revolution** и **miniShop2**, который добавляет массовые действия в дерево категорий в окне создания и редактирования опций товаров.

Компонент создан для практической задачи: в стандартном интерфейсе miniShop2 опции приходится привязывать к категориям вручную, по одному чекбоксу. На проектах с большим каталогом это неудобно и отнимает много времени. `ms2TreeHelper` добавляет быстрые действия прямо в toolbar дерева категорий.

## Возможности

В окне создания и редактирования опции miniShop2 компонент добавляет:

- массовое назначение опции на все категории
- назначение опции на все дочерние категории выбранной категории
- снятие опции со всех категорий

## Основные преимущества

- не требует правок `core` MODX
- не требует правок `core` miniShop2
- не меняет структуру базы данных
- не вмешивается в штатные процессоры miniShop2
- подключается как отдельный extra
- ускоряет работу администратора и контент-менеджера

## Как это работает

Компонент подключает собственный JavaScript в manager MODX и расширяет интерфейс стандартного окна miniShop2. Массовые действия работают поверх существующего дерева категорий и используют штатную механику выбора и сохранения категорий.

Это позволяет:
- сохранить совместимость с miniShop2
- не ломать существующие настройки
- безопасно обновлять MODX и miniShop2 без ручного переноса правок core

## Где особенно полезно

`ms2TreeHelper` особенно удобен для:

- интернет-магазинов с большим деревом категорий
- каталогов с десятками и сотнями товарных разделов
- проектов с конфигуратором товаров
- проектов, где опции регулярно добавляются и обновляются

## Совместимость

Проверялось на:

- **MODX Revolution 2.8.x**
- **miniShop2 3.x**

При необходимости компонент можно адаптировать под другие версии, если в manager отличается структура окна или toolbar дерева.

## Установка

1. Установите пакет `ms2treehelper` через **Extras → Installer**.
2. Очистите кэш MODX.
3. Перезагрузите manager.
4. Откройте **miniShop2 → Настройки → Опции**.
5. Создайте новую опцию или откройте существующую.
6. Используйте кнопки массового выбора над деревом категорий.

## Безопасность изменений

Компонент разработан так, чтобы не изменять core-файлы MODX и miniShop2. Это делает внедрение и сопровождение более безопасным и удобным.

## Почему этот компонент полезен

`ms2TreeHelper` решает небольшую, но очень частую и раздражающую проблему в miniShop2. На проектах с большим каталогом он экономит множество однотипных действий и делает работу с опциями заметно быстрее.
